Omschrijving

Tijdens mijn stage bij Wilms NV (2025) heb ik een geautomatiseerde deployment en orkestratie-oplossing ontwikkeld.
Hierbij werd een custom backend API gebouwd met FastAPI, die via Ansible Runner infrastructuur en software uitrolt.
De volledige oplossing maakt gebruik van Docker & Docker Compose voor containerisatie, Azure DevOps Pipelines voor CI/CD, en OpenBao voor secrets management.
Deze modulaire, open-source gebaseerde architectuur sluit aan bij de interne kennis en infrastructuur van Wilms NV, en biedt maximale flexibiliteit en beheersbaarheid
